## Basic Usage

Combine each size variant with any size related property such as `width`, `height`, `padding`, `margin`, and etc.

```html
<div class="w-32 h-32 ...">
  <!-- 32px width and 32px height -->
</div>

<div class="mb-24 ...">
  <!-- 20px margin-bottom -->
</div>
```
